// Features

Edit with validation

Edit variables with inline validation, autocomplete suggestions, and visual diff showing old vs new values.

Search & Filter

Search across key, value, and description. Filter by category to focus on what matters.

Snapshots

Capture your environment state to JSON. Back up, share between machines, track changes over time.

Presets

Named configurations for different workflows. Five built-in presets included. Switch contexts instantly.

Themes

Four built-in color themes: default, nord, dracula, light. Press t to cycle.

Undo / Redo

Full undo/redo with persistent history across sessions. Never lose a change.
